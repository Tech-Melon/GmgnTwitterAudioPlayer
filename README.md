# 🍉 Gmgn Twitter Audio Player (抢先体验版)

这是一个帮你在看 GMGN 网站时，只要你关注的推特大V一发文，就能第一时间听到专属提示音的小插件。

不用一直死死盯着屏幕，听到声音就知道大鱼动了，让你在交易市场里永远快人一步！

> **🚧 温馨提示：** 这个插件目前已经提交给 Chrome 官方商店等待审核啦。在商店正式上线之前，你可以按照下面的“傻瓜式”教程手动安装尝鲜！

---

## 📥 第一步：下载并安装插件（非常简单）

只需要花两分钟，把插件手动装到浏览器里：

1. **下载文件**：点击这个页面右上角的绿色按钮 **"Code"**，选择 **"Download ZIP"**，把文件下载到你的电脑上。
2. **解压文件**：把下载好的 ZIP 压缩包解压出来（建议把解压后的文件夹放在电脑里一个固定的地方，不要删掉它）。
3. **打开扩展中心**：打开你的 Chrome 浏览器，在最上面的网址栏复制并粘贴这段代码 `chrome://extensions/` 然后按回车。
4. **开启开发者模式**：在页面的**右上角**，找到“开发者模式”的开关，把它打开。
5. **安装插件**：点击页面左上角的 **“加载已解压的扩展程序”** 按钮，然后选中你刚才解压出来的那个文件夹。

搞定！现在你的浏览器右上角，应该会出现一个清新的 🍉 **西瓜猫图标**啦！把它“固定”在工具栏上会更方便哦。

---

## ⚙️ 第二步：教你怎么设置专属声音

1. **点开小西瓜**：点击浏览器右上角的 🍉 插件图标，打开设置面板。
2. **上传你喜欢的声音（可选）**：在“自定义音频库”卡片里，你可以选择电脑里自己喜欢的音乐或提示音（MP3、WAV 都可以）导入进去。
3. **给大佬绑定声音**：在“添加映射规则”卡片里，输入你想重点监控的推特账号（比如马斯克就填 `elonmusk`），然后在下拉菜单里选一个刚才上传的声音，或者选我们内置好的马斯克、CZ 专属声音。最后点击“添加映射”。
4. **调音量**：面板最上面有个滑块，觉得声音吵可以随时调小。

---

## ⚠️ 第三步：最最最关键的一步（不看没声音！）

设置好之后，在浏览器里打开 GMGN 的推特监控网页（比如 `https://gmgn.ai/follow?chain=bsc` ）。

👉 **请务必用鼠标在网页的随便哪个空白地方，点一下！**

*（为什么要这样？因为现在的浏览器为了防止垃圾网页乱播广告，默认是禁止自动播放声音的。你只要点一下网页，就是告诉浏览器：“我允许这个网页发声”，这样插件才能正常为你播报！）*

---

## 🎉 最终效果

完成上面三步，你就可以把 GMGN 的网页挂在后台，安心去干别的事情了。

只要 GMGN 网站推送了你刚才设置的那个推特大佬的新推文，插件就会在 **0.01 秒** 内“叮”地一声响起来！你给他设置了什么声音，它就播什么声音，绝对不会漏掉任何关键情报。

---

## ❓ 常见问题小白解答

**问：为什么有时候没声音，网页控制台还弹红字报错？**
答：如果你在面板里改了设置，或者更新了插件，记得一定要回到 GMGN 的网页，**按一下 `F5` 刷新页面**。刷新完别忘了再用鼠标点一下页面哦！

**问：一直响太吵了，我想暂时关掉怎么办？**
答：点开右上角的 🍉 西瓜猫图标，最上面有个开关，直接关掉它就静音啦。

**问：我换了一台新电脑，之前设置的好几十个声音和规则要重新弄吗？**
答：不用那么麻烦！在面板的“规则管理”那里，点一下“导出规则”，会下载一个小文件。你把这个文件发到新电脑上，点“导入规则”选它，瞬间就能全部恢复！
